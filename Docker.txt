For running Docker image of AIFordevs follow simple instructions below :

Docker Image Name : aifordevs 
Docker Image Command : docker run -it -p 8000:8000 aifordevs 

Steps : 
1. Install and download docker desktop in your OS .
2. Create Account on DockerHub and keep docker desktop running 
3. After open vs code and run give command docker run -it -p 8000:8000 aifordevs
